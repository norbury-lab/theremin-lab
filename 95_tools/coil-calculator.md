# コイル計算ツール

Wheeler公式による単層エアコイルの設計・検算ツールです。

巻き芯の直径・ワイヤー径・目標インダクタンスを入力すると、巻き回数と巻き幅を計算します。巻き機の設定値（Single Layer Wraps・Axis Width）も表示されます。
