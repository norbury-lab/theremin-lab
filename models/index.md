# モデル・カタログ

テルミンの主要モデルを年代順にまとめました。
各モデルの詳細ページは随時追加していきます。

---

{: .model }
> **RCA Thereminvox（1929）**
>
> 製造元：GE / Westinghouse（RCAライセンス）／ 回路方式：真空管ヘテロダイン／ 音域：3.5〜4オクターブ
>
> 世界初の商業量産テルミン。Léon Theremin自身が設計。製造台数約500台、現存約136台。

{: .model }
> **Big Briar Series 91（1991）**
>
> 製造元：Big Briar／ 回路方式：位相検出（VCO型）／ 音域：約4オクターブ
>
> Bob Moogによる復刻。91A・91B・91Cの3モデル、合計約150台。出荷前にLéon Theremin自身が一台一台チェックした。

{: .model }
> **Etherwave Standard（1996）**
>
> 製造元：Big Briar → Moog Music／ 回路方式：ヘテロダイン／ 音域：5オクターブ
>
> 最も普及したテルミン。キットから始まり、現在まで生産が続く最長寿モデル。技術資料が最も充実している。

{: .model }
> **Ethervox（1997）**
>
> 製造元：Big Briar／ 回路方式：ヘテロダイン＋VCO型／ 音域：2〜4オクターブ切替
>
> 唯一のフルMIDI対応テルミン。Bob Moogが「最高傑作」と称した。製造台数は55台前後。

{: .model }
> **Etherwave Pro（2004）**
>
> 製造元：Moog Music／ 回路方式：ヘテロダイン（強化版）／ 音域：6.5オクターブ
>
> Bob Moog最後のテルミン設計。製造台数約1,500台。現在Moogのサポート終了。

{: .model }
> **Etherwave 2022（現行）**
>
> 製造元：Moog Music（inMusic）／ 回路方式：ヘテロダイン（改良）／ 音域：5オクターブ
>
> StandardとPlusを統合した現行モデル。CV出力（Pitch/Vol/Gate）搭載。

{: .model }
> **Theremini（2014）**
>
> 製造元：Moog Music／ 回路方式：デジタル（Animoog派生）／ 内蔵スピーカーあり
>
> ピッチ補正機能付き。入門向け。USB MIDI対応。

{: .model }
> **Claravox Centennial（2020）**
>
> 製造元：Moog Music／ 回路方式：デジタル＋アナログ
>
> Léon Theremin生誕100周年記念モデル。限定生産。
